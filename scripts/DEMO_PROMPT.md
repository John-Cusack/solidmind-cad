# Demo Prompt

Paste this into Claude Code with FreeCAD and Isaac bridge running.

---

Build a full 18-DOF hexapod — 3 joints per leg: coxa, femur, tibia. Rectangular chassis ~120×90mm, 5mm thick. Leg segments: coxa 52mm, femur 66mm, tibia 133mm. Servos are AX-12A (32×24×24mm). Standing pose: femur 30° below horizontal, tibia 70°. After building, define the mechanism, validate it, export a URDF sim package with 158mm ground clearance, then import into Isaac Sim and make it walk forward.
