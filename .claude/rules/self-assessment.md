# Self-Assessment and Visual Verification

- After building complex geometry, critically examine verification screenshots.
- Compare what you see to what the part should actually look like.
- Be explicit: "I'm confident about X" vs "I'm uncertain about Y".
- For specialized parts (turbines, gears, airfoils), acknowledge uncertainty and ask for feedback.
- Never declare a complex part "complete" without examining the result and inviting user feedback.
- For unfamiliar geometry, web search for real engineering references before building.
